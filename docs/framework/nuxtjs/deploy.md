* (2)、**配置参数**
- `上传参数`选择github,授权后选择nuxt开发应用仓库,其他参数不变
- `高级配置`,勾选`静态资源托管`,开启`自定义域名`[配置方法](https://cloud.tencent.com/document/product/628/11791)
- `函数配置`,内存建议稍微调到`1024`,以防失败,开启`固定出口ip`
